# David Motsoeneng - Portfolio

Personal portfolio website showcasing my work as a Backend Engineer and Systems Thinker.

## 🚀 Live Site

Visit: [https://thedirectorza.github.io/](https://thedirectorza.github.io/)

## 🛠️ Built With

- React
- SASS
- Lottie Animations
- GitHub Pages

## 📝 Setup

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```
